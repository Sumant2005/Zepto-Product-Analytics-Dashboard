# Zepto Product Analytics Dashboard

This project analyzes product performance, pricing, discounts, and inventory insights using Power BI.

## Dataset
The dataset contains information about:
- Product categories
- Product prices and discounts
- Inventory availability
- Product quantity

Total records: 3732 products.

## Key Metrics
- Total Products
- Total Revenue
- Average Discount %
- Out of Stock Count
- Inventory Value

## Dashboard Insights
The dashboard provides insights on:

• Sales volume by product category  
• Top 10 best-selling products  
• Revenue contribution by category  
• Out-of-stock products by category  
• Average discount offered across categories  

## Tools Used
- Power BI
- Data Cleaning
- Data Visualization
